This is the game which I created as part of a team during my first year in university. My team had 4 members.

My main resposibilities in the team were programming, art and SFX.

I wrote almost all code for gameplay mechanics. I created all the vector art in the game, which is anything except for the actual level.

{% include gallery id="gallery_wool1" layout="half" thumb_height="200px"%}

All characters were drawn using a vector art program and were then exported into PNG's and rigged in Unity.
I created many animations for the characters, using animation events to sync SFX.
